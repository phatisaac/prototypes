# prototypes

Description: This repository contains the conceptual Python workflows designed to modernize the post-TB surveillance system at MODVA. These scripts represent the logic currently being transitioned from manual/Excel-based registries to automated Python pipelines.
model_benchmark.py: A proof-of-concept script demonstrating the benchmarking of Random Forest vs. Linear Regression for predicting post-TB quality of life.

spatial_registry_pipeline.py: A prototype ETL pipeline for linking patient clinical outcomes with village-level geospatial data to automate referral generation.
